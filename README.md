# Overview
In this project am gonna try to make a fingerprint matching pattern between two image to see how the match of these fingerprint like a police fingerprint matching system, this may include some of a image processing knowledge about feature mathcing.
There are plenty of a feature matching technique for example ORB, SIFT, FLANN. In this repo am gonna try to find the best method that suit this project. <br><br>
